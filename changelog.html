Changelog

[1.0.0] - Initial Release

  • Core theme functionality and setup.
  • Documentation Links: Setup Guide, Features Overview, and Changelog.

[1.1.0] - Header HTML Compliance

  • Added JSON-LD Support: Implemented structured data for SEO across product, collection, blog, contact, and homepage templates.
  • Integrated Dynamic Social Media Links: Automatically generated sameAs links for social platforms based on Shopify theme settings. Supported platforms include Facebook, Instagram, YouTube, TikTok, Pinterest, Snapchat, Tumblr, and Vimeo.
  • Added Meta Tags for SEO: Enhanced search engine indexing and multilingual support for meta tags.

[1.2.0] - Vue.js, GSAP, and LazySizes Configuration

  • Vue.js: Configured for Shopify compatibility to enhance interactivity for merchant stores.
  • GSAP: Implemented animations to enhance user experience through dynamic transitions.
  • LazySizes: Integrated for optimized image loading using data-src attributes to boost performance by deferring non-visible images.

[1.3.0] - Typography

  • Added:
    • Responsive Typography System:
      • Support for all heading levels (H1â€“H6) and body text.
      • Customizable fonts, sizes, line heights, and weights via the Theme Editor.
      • Default golden ratio-based scaling for harmonious design.
  • Updated: Improved text readability with responsive line heights and word-breaking settings.

[1.4.0] - Colors

  • Added:
    • Global Colors Settings: Customizable options for body background color, body text color, heading text color, and paragraph text color with default fallbacks.
    • Text Alignment: New setting to control alignment (left, center, right, justify).
    • Responsive Design: Defined breakpoints for mobile, tablet, and desktop views to ensure layout consistency.
    • SCSS Enhancements: Linked new global color settings to CSS variables for reusable styling and added dynamic breakpoints.

[1.5.0] - Logo and page width

  • Added:
    • Logo Settings: New settings for uploading a custom logo, including width adjustment, custom URL linking, and alt text for accessibility.
    • Alignment Options: Ability to align the logo to the left, center, or right using a dynamic selector.
    • Aspect Ratio Handling: Automatic height adjustment based on a 1.8:1 aspect ratio for proportional scaling.
    • Lazy Loading: Integrated lazy loading for improved performance and page speed optimization.
    • SCSS Enhancements: Updated styles for responsive logo display with breakpoints for mobile, tablet, and desktop views.
    • Page width: allows you to customize the content width for desktop screens, ensuring optimal layout and readability.
