Changelog

[1.0.1] - Header HTML Compliance

Added

  • Added JSON-LD Support: Implemented structured data for SEO across product, collection, blog, contact, and homepage templates. Ensures compatibility with Google Rich Results Test and improves search visibility. Includes Translation Support: JSON-LD fields now support multilingual themes via en.default.schema.json entries.
  • Integrated Dynamic Social Media Links: Dynamically generated sameAs links for social platforms based on Shopify theme settings. Supported platforms include Facebook, Instagram, YouTube, TikTok, Pinterest, Snapchat, Tumblr, and Vimeo. Localized Labels and Descriptions: Social account settings now include translations for UI clarity.
  • Added Meta Tags for SEO: <meta name="robots" content="index, follow"> ensures pages are indexed and links followed by search engines. <link rel="icon"> dynamically references the favicon set in theme settings. Meta tag descriptions and labels are translation-ready.
