Customization Instructions

Learn how to customize the EcoFlow Shopify theme to suit your needs.

Available Options

  • Change color schemes.
  • Add custom branding elements.
  • Modify layout settings.

Detailed customization steps will be available soon.

Managing Social Media Links

  1. Navigate to Theme Settings > Social Accounts in the Shopify admin.
  2. Add your profile URLs (e.g., Facebook, Instagram, Twitter). Supported platforms include:
    • Facebook, Instagram, YouTube, TikTok, Pinterest, Snapchat, Tumblr, Vimeo, and more.
  3. Each social media field includes:
    • Label: E.g., “Facebook Profile URL.”
    • Info Tooltip: Guidance on what to enter (e.g., "Enter the full URL of your Facebook profile").
  4. Translation Support: Labels and tooltips for social media fields are defined in en.default.schema.json under social_accounts.

Setting the Favicon

  1. Go to Theme Settings > Logo.
  2. Upload your desired favicon under "Favicon."
  3. The system dynamically generates the <link rel="icon"> tag: 
    <link rel="icon" type="image/png" href="{{ settings.favicon | image_url: width: 32, height: 32 }}">
  4. Translation Support: The "Favicon" label and info text are defined in en.default.schema.json under settings_schema.logo.

Social Media Metadata

  1. Go to Theme Settings > Social Meta to define:
    • Meta Title (e.g., "Default Meta Title").
    • Meta Description (e.g., "A short description for social media previews").
    • Meta Image (used for sharing on platforms like Facebook or Twitter).
  2. Translation Support: Labels and descriptions for these fields are defined in en.default.schema.json under settings_schema.social_meta.

Using JSON-LD

JSON-LD is automatically included based on the page type:

  • Product Pages: Includes structured data for product details, brand, offers, and social links.
  • Collection Pages: Adds breadcrumb navigation data.
  • Blog Pages: Includes metadata for blog articles.
  • Index and Contact Pages: Includes structured data for WebPage or ContactPage.

Responsive Customization

The EcoFlow Shopify theme includes centralized breakpoints for responsive styling and animations.

  • Breakpoints are defined in a single `breakpoints.js` file for consistency across styles and animations.
  • Modify breakpoints in one place to update both SCSS and GSAP animations.

Breakpoints Overview

  • Mobile: max-width: 768px
  • Tablet: max-width: 1024px
  • Desktop: min-width: 1025px

Typography Customization

The EcoFlow theme allows full control over typography directly in the Theme Editor.

Available Settings

  • Body Font: Select from system fonts, Google Fonts, or upload a custom font.
  • Heading Font: Choose a separate font for headings (H1–H6).
  • Font Sizes: Adjust sizes for body text and individual heading levels:
    • H1 Font Size: Default 32px, adjustable 20px–72px.
    • H2 Font Size: Default 25px, adjustable 20px–48px.
    • Body Font Size: Default 16px, adjustable 12px–24px.
  • Line Heights: Customize line spacing for headings and body text (1.2–2.0).
  • Font Weights: Select weights from 100 to 900.

How to Customize

  1. Open the Theme Editor in Shopify.
  2. Navigate to Typography Settings.
  3. Adjust the following:
    • Body and heading fonts.
    • Sizes for body text and headings (H1–H6).
    • Line heights for body text and headings.
    • Font weights for body and headings.

Preview Changes

Use the responsive preview tool to ensure the typography adjusts well across devices.
