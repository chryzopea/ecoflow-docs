Setting the Favicon

  1. Navigate to Theme Settings > Logo Section in the Shopify admin.
  2. Under Upload Favicon, click Select to upload your desired favicon file.
  3. Ensure that the favicon meets the recommended size and format requirements:
    • Size: 32x32 pixels (recommended).
    • Format: PNG.
  4. Translation Support: The "Upload Favicon" label and info text are defined in en.default.schema.json under settings_schema.logo.

Note: If no favicon is uploaded, the theme will fall back to the default favicon: default-favicon.png.

Managing Social Media Links

  1. Navigate to Theme Settings > Social Media Accounts in the Shopify admin.
  2. Add your profile URLs in the respective fields, replacing the placeholder "N/A" with the full URL of your social media profile. Supported platforms include:
    • Facebook, Instagram, YouTube, TikTok, Pinterest, Snapchat, Tumblr, Vimeo, X (formerly Twitter), and more.
  3. Each social media field includes:
    • Label: E.g., “Facebook Profile URL.”
    • Placeholder: Default text ("N/A") indicating where to paste the URL.
  4. Translation Support: Labels for social media fields are defined in en.default.schema.json under social_accounts.

Note: If links don’t appear, ensure URLs are entered correctly and saved.

Social Media Metadata

  1. Navigate to Theme Settings > Social Media Metadata in the Shopify admin.
  2. Define the following fields:
    • Meta Title: The title that will appear on social media shares (default: "Default Meta Title").
    • Meta Description: A short description that will appear on social media shares (default: "Default Meta Description").
    • Meta Image: Upload an image that will appear on social media shares. Use the Select button to upload or Explore free images.
  3. Ensure that the uploaded image meets the recommended dimensions and format:
    • Format: PNG or JPEG.
  4. Translation Support: Labels and descriptions for these fields are defined in en.default.schema.json under settings_schema.social_meta.

Tip: Ensure your Meta Title and Description are concise yet descriptive to improve visibility and engagement on social media platforms.

Using JSON-LD

JSON-LD is automatically included in the EcoFlow theme to provide structured data for different page types. This enhances SEO and ensures compatibility with rich results on search engines.

  • Product Pages:
    • Structured data includes product details like name, description, featured image, price, currency, availability, SKU, and brand.
    • If applicable, includes aggregate ratings (ratingValue and reviewCount).
    • Includes social media links in the sameAs property if defined.
  • Collection Pages:
    • Includes breadcrumb navigation with hierarchical relationships between pages.
    • Provides collection title, description, and collection image.
    • Includes social media links in the sameAs property if defined.
  • Blog Pages:
    • Structured data for blog posts includes headline, featured image, author, publication date, and excerpt (description).
    • Identifies the blog post publisher with a logo and name.
    • Includes social media links in the sameAs property if defined.
  • Index and Contact Pages:
    • Structured as @type: WebPage or @type: ContactPage.
    • Includes page name, description, and URL.
    • Includes social media links in the sameAs property if defined.

Social Media Integration: Social media links are dynamically included in the sameAs property for all page types if they are defined in the theme settings and are not left blank or set to "N/A."

Publisher Metadata: All pages include publisher details, such as the store name and logo, enhancing brand recognition.

Note: Validate the structured data using Google Rich Results Test to ensure it meets SEO and rich snippet standards.

Responsive Customization

The EcoFlow Shopify theme includes centralized breakpoints for responsive styling and animations.

  • Breakpoints are defined in a single breakpoints.js file for consistency across styles and animations.
  • Modify breakpoints in one place to update both SCSS and GSAP animations.

Breakpoints Overview

  • Mobile: max-width: 749px
  • Tablet: max-width: 990px
  • Desktop: min-width: 1200px

Tip: Test your changes on multiple devices to ensure consistent responsiveness.

Typography Customization

The EcoFlow theme offers comprehensive typography customization options, enabling you to tailor text styles to match your brand.

Available Settings

  • Global Body Typography: Adjust the base font for all text elements, including buttons, links, and general text. Customize:
    • Body Font Family: Choose from system fonts, Google Fonts, or upload a custom font.
    • Body Font Size, Weight, Line Height, and Letter Spacing.
    • Text Alignment: Align text to left, center, right, or justify.
  • Headings Typography: Customize the appearance of headings (H1–H6):
    • Heading Font Family: Set a font specific to headings, overriding the body font.
    • Font Sizes, Line Heights, and Weights for each heading level.
    • Letter Spacing and Margins (top and bottom) for all headings.
  • Paragraph Typography: Define styles specific to paragraphs:
    • Font Size, Weight, Line Height, and Letter Spacing.
    • Text Alignment: Set alignment for paragraph text independently.
    • Paragraph Margins (top and bottom): Control spacing around paragraphs.

How to Customize

  1. Open the Theme Editor in Shopify.
  2. Navigate to Typography Settings.
  3. Adjust the settings for body, headings, and paragraphs as needed.

Preview Changes

Use the responsive preview tool to ensure the typography looks consistent across devices.

Color Customization

  • Global Colors: Navigate to the Theme Editor under Global Colors. Adjust the settings for:
    • Body Background Color
    • Body Text Color
    • Heading Text Color
    • Paragraph Text Color
    Changes will reflect dynamically across the theme.

Preview Changes

Use the responsive preview tool to ensure the typography adjusts well across devices.

Body Width Customization

The EcoFlow theme allows you to customize the content width for desktop screens, ensuring optimal layout and readability.

Available Setting

  • Body Width (px): Adjust the overall content width for desktop views. The default value is 1200px, with a customizable range to suit your design preferences.

How to Customize

  1. Open the Theme Editor in Shopify.
  2. Navigate to Theme Settings > General Settings.
  3. Adjust the Body Width slider to your desired value or enter a specific pixel value in the input box.

Preview Changes

Use the responsive preview tool to verify the content width across devices and ensure it meets your design needs.

Customizing the Store Logo

  1. Navigate to Theme Settings > Logo Section in the Shopify admin.
  2. Under Upload Logo, click Select to upload your desired logo file:
    • Format: PNG or JPEG.
    • Recommended Maximum Width: 450px.
    • File Size: Ensure the file size is under 1MB for optimal performance.
  3. Set the Alt Text:
    • In the Alt Text for Logo field, provide a brief description of your logo.
    • Alt text is important for accessibility and SEO.
    • If left blank, your shop name will be used as the default alt text.
  4. Set a Custom URL:
    • Enter a custom URL for your logo in the Custom Logo URL field (e.g., a campaign page).
    • Leave this field blank to link the logo to your homepage.
  5. Adjust the Logo Width:
    • Use the Logo Width slider to adjust the logo's width to your preference.
    • The height will automatically adjust to maintain a 1.8:1 aspect ratio.
  6. Align the Logo:
    • Select your preferred alignment (Left, Center, or Right) from the Logo Alignment dropdown.

Note: All settings, including labels and info text, are translatable. The translation keys are defined in en.default.schema.json under settings_schema.logo.
