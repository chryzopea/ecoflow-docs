Customization Instructions

Learn how to customize the EcoFlow Shopify theme to suit your needs.

Available Options

  • Change color schemes.
  • Add custom branding elements.
  • Modify layout settings.

Detailed customization steps will be available soon.

Vue.js Integration

  1. Add components and logic to scripts.js.
  2. Mount Vue apps on Shopify elements: 
    const app = createApp(App);
app.mount('#vue-app');
  3. Test Vue rendering with a static or dynamic template.

GSAP Animations

  1. Define animations using gsap.to(): 
    gsap.to('.element', { x: 100, duration: 1 });
gsap.to('.another-element', { opacity: 1, duration: 1.5 });
  2. Ensure target elements exist in the DOM.
