Troubleshooting

If you encounter issues with the EcoFlow theme, follow these steps to resolve common problems or contact support if needed.

Troubleshooting Guidelines

Problem: Social media links missing

Solution: Verify profile URLs in Theme Settings > Social Accounts. Ensure they are entered correctly.

Problem: Favicon not displaying

Solution: Check if the favicon is uploaded in Theme Settings > Logo. Verify the correct favicon reference in the browser's developer tools under the <head> section.

Problem: Font not applying

Solution: Ensure the correct font is selected in the Typography section of the Theme Editor. Verify that the uploaded font files are in compatible formats (.woff, .woff2, .ttf).

Problem: Vue.js app not rendering

Solution:

  1. Ensure the mounting DOM element exists in your Liquid file (e.g., #vue-app).
  2. Verify that Vue is properly initialized and no syntax errors exist in the script.

Problem: Images not loading (LazySizes)

Solution:

  1. Verify that data-src paths are correct in your image tags.
  2. Check the browser console for LazySizes initialization errors.

Problem: GSAP animations not triggering

Solution:

  1. Ensure target elements exist in the DOM (e.g., .element, .another-element).
  2. Debug animation logic in your scripts.js file using browser developer tools.

Problem: Responsive breakpoints not working

Solution:

  1. Ensure the breakpoints.js file is correctly imported into your GSAP scripts.
  2. Check SCSS compilation logs for syntax errors related to breakpoints.
  3. Use browser developer tools to verify that media queries are applied correctly.

Problem: Text appears too small on mobile

Solution:

  1. Check the responsive typography settings in your styles.scss file.
  2. Ensure mobile-specific media queries are implemented correctly in your SCSS files.

Problem: Logo doesn't display properly

Solution:

  1. Check the file format:
    • Ensure the logo is in PNG or JPEG format. Other formats like SVG are not supported in the Theme Editor.
  2. Verify the upload:
    • Confirm the logo is uploaded and saved in the Theme Editor.
  3. Check file size or resolution:
    • Ensure the file size is under 1MB and the width does not exceed 450px.
  4. Ensure alt text is present:
    • If the alt text is missing, add a description in the "Alt Text for Logo" field in the Theme Editor.

Problem: Custom logo URL not working

Solution:

  1. Verify the custom URL is entered correctly in the "Custom Logo URL" field.
  2. Ensure the URL starts with https://.

Problem: Logo alignment looks off

Solution:

  1. Check the "Logo Alignment" setting in the Theme Editor.
  2. Adjust the alignment as needed (Left, Center, or Right).

Problem: Logo scaling issues on mobile

Solution:

  1. Ensure the logo file is responsive.
  2. Use the Theme Editor's preview feature to check how it appears on mobile.
