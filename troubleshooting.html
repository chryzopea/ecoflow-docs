Troubleshooting

If you encounter issues with the EcoFlow theme, follow these steps to resolve common problems or contact support if needed.

Troubleshooting Guidelines

Problem: Social media links missing

Solution: Verify profile URLs in Theme Settings > Social Accounts. Ensure they are entered correctly.

Problem: Favicon not displaying

Solution: Check if the favicon is uploaded in Theme Settings > Logo. Verify the correct favicon reference in the browser's developer tools under the <head> section.

Problem: Font not applying

Solution: Ensure the correct font is selected in the Typography section of the Theme Editor. Verify that the uploaded font files are in compatible formats (.woff, .woff2, .ttf).

Problem: Background image or video appears faded

Solution:

  1. Navigate to the "Background Settings" in the Theme Editor.
  2. Set the Background Opacity to 0 for full visibility of the image or video.
  3. Ensure the Background Color is set to transparent if no overlay is desired.

Problem: Section width not changing

Solution:

  1. Verify the "Section Width" setting in the Theme Editor.
  2. Ensure CSS variables for width are applied correctly in your custom styles or overrides.

Problem: Section spacing not applying

Solution:

  1. Ensure the "Use Local Margin Overrides" option is enabled for the section.
  2. Verify that padding and margin values are set in the Theme Editor for both desktop and mobile views.

Problem: Fallback image for video not loading

Solution:

  1. Ensure a fallback image is uploaded in the "Mobile Video Fallback Image" field.
  2. Verify that the image format is either JPEG or PNG and is correctly sized.

Problem: Borders not visible

Solution:

  1. Ensure border width is greater than 0px.
  2. Verify the border style is not set to "none" in the section settings.
  3. Confirm that the border color is not transparent.

Problem: Typography alignment not reflecting

Solution:

  1. Check the "Content Alignment" setting in the Theme Editor.
  2. Inspect the CSS for potential conflicts with other styles or customizations.

Problem: Dynamic blend modes not working

Solution:

  1. Ensure the blend mode selected in the Theme Editor is supported by your target browsers.
  2. Verify that the background image or color is properly configured to interact with the blend mode.

Problem: Mobile-specific backgrounds not displaying

Solution:

  1. Ensure the "Enable Mobile Background" or "Enable Mobile Background Image" settings are active.
  2. Preview the mobile view in the Theme Editor to confirm the settings are applied.

Problem: Changes not visible after saving

Solution:

  1. Verify that the changes are saved in the Theme Editor.
  2. Clear the browser cache and refresh the page.
  3. Check the Shopify admin for any errors in saving the theme settings.
