Common Issues

Vue.js Issues

  • Problem: Vue app not rendering.
    Solution: Ensure the mounting DOM element (e.g., #vue-app) exists. Verify Vue initialization and template correctness.

LazySizes Issues

  • Problem: Images not loading.
    Solution: Check data-src paths for correctness. Confirm LazySizes initialization in the console.

GSAP Issues

  • Problem: Animations not triggering.
    Solution: Verify that target elements (.element, .another-element) are present in the DOM. Debug animation logic in scripts.js.

Social Media Links Missing

  • Ensure the URLs are correctly entered in Theme Settings > Social Accounts.
  • Check the translation file for proper id and label mappings.

Favicon Not Displaying

  • Verify the favicon is uploaded in Theme Settings > Logo.
  • Confirm the generated <link rel="icon"> tag in the HTML <head>.

Translation Issues

  • Ensure the en.default.schema.json contains all necessary entries under settings_schema.
  • Example:
    • {
      "settings_schema": {
        "social_accounts": {
          "facebook": {
            "label": "Facebook Profile URL",
            "info": "Enter the full URL of your Facebook profile."
          }
        }
      }
    }

Responsive Breakpoints Not Working

  • Problem: Breakpoints are not applied correctly.
  • Solution:
    • Ensure the breakpoints.js file is imported correctly in your GSAP scripts.
    • Check SCSS compilation for breakpoint syntax errors.
    • Use browser developer tools to confirm correct media query matching.

Troubleshooting Typography Issues

1. Font Not Applying

Cause: The selected font may not be loaded correctly.

Solution:

  • Ensure the font is uploaded in the correct format (.woff, .woff2, .ttf).
  • Verify the font is selected in the Theme Editor.

2. Text Appears Too Small on Mobile

Cause: Mobile font scaling may not be applied.

Solution:

  • Check the responsive typography settings in styles.scss.
  • Ensure mobile-specific media queries are implemented correctly.

3. Line Length Too Long

Cause: max-width for paragraphs is missing.

Solution:

  • Verify max-width: 70ch; is applied to p elements in _typography.scss.

4. Font Weight Not Updating

Cause: Font weight settings may not be linked to variables.

Solution:

  • Check that variables.$body-font-weight and variables.$heading-font-weight are correctly referenced in _typography.scss and styles.scss.

Troubleshooting COLOR Issues

  • Colors Not Applying:

    Verify that the color settings in the Theme Editor are non-blank. If issues persist, check for typos in the SCSS variables or the CSS `:root` declarations.

  • Paragraph Alignment Issues:

    Ensure the alignment settings are saved in the Theme Editor. Check for conflicting CSS styles in custom sections.

  • Responsive Layout Problems:

    Ensure the correct breakpoints are used:

    • Mobile: max-width: 749px
    • Tablet: max-width: 990px
    • Desktop: min-width: 1200px

  • SCSS Variables Not Reflecting:

    Ensure your SCSS file is compiled and that the changes are included in the bundled CSS file deployed to Shopify.
