Common Issues

Vue.js Issues

  • Problem: Vue app not rendering.
    Solution: Ensure the mounting DOM element (e.g., #vue-app) exists. Verify Vue initialization and template correctness.

LazySizes Issues

  • Problem: Images not loading.
    Solution: Check data-src paths for correctness. Confirm LazySizes initialization in the console.

GSAP Issues

  • Problem: Animations not triggering.
    Solution: Verify that target elements (.element, .another-element) are present in the DOM. Debug animation logic in scripts.js.

Social Media Links Missing

  • Ensure the URLs are correctly entered in Theme Settings > Social Accounts.
  • Check the translation file for proper id and label mappings.

Favicon Not Displaying

  • Verify the favicon is uploaded in Theme Settings > Logo.
  • Confirm the generated <link rel="icon"> tag in the HTML <head>.

Translation Issues

  • Ensure the en.default.schema.json contains all necessary entries under settings_schema.
  • Example:
    • {
          "settings_schema": {
            "social_accounts": {
              "facebook": {
                "label": "Facebook Profile URL",
                "info": "Enter the full URL of your Facebook profile."
              }
            }
          }
        }

Responsive Breakpoints Not Working

  • Problem: Breakpoints are not applied correctly.
  • Solution:
    • Ensure the `breakpoints.js` file is imported correctly in your GSAP scripts.
    • Check SCSS compilation for breakpoint syntax errors.
    • Verify browser developer tools to confirm correct media query matching.
