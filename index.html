EcoFlow Documentation

Welcome to EcoFlow Documentation

1. Introduction

EcoFlow is a flexible theme designed for small and medium product catalogs. It offers multiple layout choices, modern product displays, and clear controls for store design and content.

2. Metafields

2.1 Product and Variant Metafields

Metafields let you show additional product information. Create metafields in Settings → Custom data, then fill them in on each product or variant. Sections and blocks display metafield content only when a corresponding field is selected in the Theme Editor.

2.2 File Metafields

Upload files in Content → Files. Select the file inside the metafield. Provide a short description so customers understand the download link.

2.3 Metafield-Supported Sections and Blocks

  • Product comparison
  • Highlights
  • Featured product
  • Info board
  • Icon + metafield blocks

2.4 Page Metafield (Mega Menu Image)

A page metafield can provide an image for the mega menu. When added, the image appears beside the menu item on hover. If no image is provided, the theme may use the first available relevant image.

3. Product Templates

3.1 Variant Selectors

Choose the variant selector style in the product settings. Selections update automatically when variants include their own images, colors, or metafield information.

3.2 Product Media

Product images, videos, and 3D models appear in the gallery. The gallery is provided through the “Product Details” sections of the product template.

3.3 Pickup Availability

Appears when enabled in Shopify Admin and the block is added in the Product Details section.

3.4 Selling Plans

Subscription options appear when your subscription app provides its block inside the Product Details section.

4. Collection Templates

4.1 Filtering

Uses Shopify’s built-in filters. Filters open in a modal on mobile. On desktop, filter controls appear as a toolbar or sidebar depending on the section settings. Active filters appear next to the product grid. Products update after filters are applied.

4.2 Load More / Pagination

If enabled, customers can load more products without leaving the page.

4.3 Collection Media

Product images, product labels, and metafield elements appear in grids and slideshows.

5. Search, Blog, Cart, Gift Card

5.1 Search

Search opens from the header. Results can include products, pages, and articles.

5.2 404 Page

You can edit the message shown on the 404 page.

5.3 Blog and Article

Blog and article pages follow your content and the theme’s global typography and layout settings.

5.4 Cart

Mini cart: Opens as a drawer with adjustable styling, text, and notifications.
Cart page: Shows products, prices, selling plans, and optional cart notes.

5.5 Gift Card

The template displays the gift card code and balance.

6. Theme Settings

6.1 General

Controls for placeholder images and badge text.

6.2 Layout

Controls for global spacing, corner rounding, and column layouts. Sections may offer local overrides.

6.3 Logo and Favicon

Upload your logo and favicon. Adjust logo size and description text.

6.4 Cart Settings

Controls the text, appearance, and notifications of the mini cart.

6.5 Social Sharing Metadata

Controls how your store appears when shared on social platforms.

6.6 Social Links

Add your social profiles. Icons appear automatically in supported areas such as the footer.

6.7 Typography

Choose fonts for headings and body text. Adjust size and scale.

6.8 Global Colors

Controls for background, text, accents, and surfaces across the theme.

6.9 Buttons

Styles for primary, secondary, add-to-cart, and checkout buttons.

6.10 Global Icons

Controls for default icon size and color. Some sections include their own overrides.

6.11 Product Labels

Create up to five label presets. Assign each preset to a product tag. Choose the text, color, shape, and position.

6.12 Back-to-Top Button

Enable and style the button.

6.13 Variant Selector Styling

Controls the unified appearance of all product variant selector types.

6.14 Filters

Adjust the appearance of filter layouts used on collection and search pages.

6.15 Search

Adjust the appearance of the search interface.

6.16 Cookie Banner

Enable and style the banner.

6.17 Custom CSS

Add custom styles to the theme.

7. Header Group

7.1 Mobile Header

Controls for logo, menu icon, search, cart, and an optional additional link. Choose between available mobile header styles.

7.2 Mobile Menu

Choose the menu and the menu style. Adjust text appearance.

7.3 Desktop Header

Choose the main and secondary menus. Choose fixed or scrolling behavior. Optional underline animation. Optional additional link.

8. Shared Section Settings

Most sections include standard design controls:

  • Width
  • Height
  • Alignment
  • Background color, image, video, overlays
  • Padding and margin
  • Borders
  • Corner rounding
  • Custom class
  • Custom CSS

9. Individual Sections

Slideshow Sections — Shared Controls

The following controls appear in all slideshow-based sections:

  • Autoplay, delay, loop, pause on hover
  • Slides per view
  • Transition effects (when available)
  • Pagination settings
  • Prev/next arrow settings

9.1.1 — 9.19 Section Summaries

Collection sections, featured products, recommendations, comparison sliders, galleries, column layouts, tabs, highlights, collection mains, product details, hero, banner, breadcrumbs, collection lists, custom Liquid, custom section, contact page, password page, page template, gift card template, info board, featured collection — each section lists only unique functional settings and omits purely visual controls.

10. Media

Images and videos adapt to available space in each section. Media loads as customers scroll.

11. Accessibility

Includes focus states, descriptive labels, keyboard navigation, and color controls that support readability. Typography and color settings help maintain accessible layouts.

12. Performance

Uses flexible media and efficient loading to support fast page performance.

13. Footer

13.1 Blocks

  • Follow on Shop
  • Icons
  • Menu
  • Payment icons
  • Social media icons

13.2 Localization

Country and language selectors appear when your store uses multiple markets or languages.

14. Blocks

Documentation includes all block types: rich text, icon with text, separator, image, video (landscape, square, portrait), custom Liquid, custom HTML, buttons (single, double), footer blocks, header blocks, app blocks, and all product-specific blocks.

Each block lists only functional settings. All purely visual fields are omitted.
