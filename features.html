Features Overview

The EcoFlow Shopify theme is designed to:

  • Provide an optimized, fast-loading design.
  • Support customization for various industries.
  • Include features for developers and merchants alike.

A detailed feature list will be added in future updates.

Key Features

  • Vue.js Integration: Lightweight JavaScript framework for dynamic Shopify content. Allows modular and reusable component-based development.
  • GSAP Animations: Smooth animations for interactive elements. Enhance user experience with dynamic transitions.
  • LazySizes for Images: Reduces page load time with lazy loading. Ensures images load only when visible in the viewport.
  • SEO Enhancements:
    • Dynamic JSON-LD supports rich snippets in search engines.
    • Optimized meta tags for better indexing and sharing.
  • Social Media Integration:
    • Dynamically generates sameAs links from theme settings.
    • Supports multilingual labels and descriptions via en.default.schema.json.
  • Favicon Support:
    • Dynamically generated <link rel="icon"> tag ensures correct favicon display.
  • Multilingual Theme Support:
    • All labels and tooltips for theme settings (social media, meta, favicon) are fully translatable.
  • Centralized Responsive Breakpoints:
    • Breakpoints are shared between SCSS and GSAP.
    • Defined in a single `breakpoints.js` file for maintainability.
    • Responsive animations and styles adapt seamlessly across devices.
  • Responsive Typography System:
    • Supports all heading levels (H1â€“H6) for semantic HTML and SEO.
    • Body text optimized for readability with a line length of 70 characters.
    • Dynamic font sizing adjusts fluidly across mobile, tablet, and desktop.
    • Customizable fonts, sizes, line heights, and weights via the Theme Editor.
    • Golden ratio-based scaling for a visually harmonious design.
  • Color System: Global Styling: Customizable color settings for body background, text, headings, and paragraphs ensure cohesive branding.
  • Paragraph Alignment Control: Fine-tune text alignment for improved readability and aesthetics.
  • Responsive Design: Centralized breakpoints for mobile, tablet, and desktop devices, ensuring seamless adaptability.
  • SCSS Integration: Variables and mixins for reusable, scalable, and maintainable styling.
