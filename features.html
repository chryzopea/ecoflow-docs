Features Overview

The EcoFlow Shopify theme is designed to help merchants create fast, customizable, and feature-rich online stores while providing developers with powerful tools for flexibility and scalability.

Performance

  • LazySizes for Images: Boosts page load speeds by loading images only when visible in the viewport, improving user experience and reducing bounce rates.
  • GSAP Animations: Smooth, dynamic animations for sliders, buttons, and other interactive elements to engage users.

Design Flexibility

  • Responsive Typography System:
    • Fully customizable fonts, sizes, and weights for body text and headings (H1â€“H6).
    • Fluid font scaling ensures consistent readability across all devices.
    • Golden ratio-based scaling delivers visually harmonious text layouts.
  • Global Color System: Adjust text, background, and heading colors through the intuitive Theme Editor.
  • Advanced Section Customization:
    • Toggle between Full Width and Body Width layouts for any section.
    • Set local spacing overrides for margins and padding on desktop and mobile devices.
    • Configure section height with options for auto, custom pixel heights, or full-window height for immersive designs.
    • Enhanced background settings with mobile-specific adjustments:
      • Upload images with customizable size, position, and blend modes.
      • Set default and mobile-specific colors with adjustable opacity.
      • Add video backgrounds for desktop, with conditional fallbacks for mobile:
        • Background videos will not load on mobile to save bandwidth and ensure browser compatibility.
        • A static fallback image can be displayed on mobile devices for a consistent user experience.
    • Customizable borders with adjustable width, style, and color for the top and bottom edges of sections.
    • Dynamic grid gap settings to control spacing between elements within sections, with options to override global grid gaps locally.
    • Custom CSS Class for Sections: Add unique classes for advanced styling or scripting, empowering developers to extend the design beyond default capabilities.
  • Dynamic Blend Modes: Choose how background images and colors interact, allowing for visually stunning designs with options like `multiply`, `overlay`, and more.

SEO and Social Media Integration

  • JSON-LD Support: Automatically adds structured data for products, collections, blogs, and pages to boost search engine visibility and support rich snippets.
  • Meta Tag Optimization: Pre-configured meta tags streamline search engine indexing and improve content sharing across platforms.
  • Social Media Integration:
    • Dynamically generates sameAs links for social platforms based on theme settings (e.g., Facebook, Instagram, TikTok).
    • Ensures accurate and engaging previews on social platforms with multilingual support.
