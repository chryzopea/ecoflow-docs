Features Overview

The EcoFlow Shopify theme is designed to help merchants create fast, customizable, and feature-rich online stores while providing developers with powerful tools for flexibility and scalability.

Performance

  • LazySizes for Images: Faster page loading by loading images only when visible in the viewport. This enhances user experience and reduces bounce rates.
  • GSAP Animations: Smooth, dynamic animations for sliders, buttons, and other interactive elements to engage users.

Design Flexibility

  • Responsive Typography System:
    • Customizable fonts, sizes, and weights for body text and headings (H1–H6).
    • Fluid font sizing adapts to different screen sizes, ensuring readability across devices.
    • Golden ratio-based scaling for visually harmonious text design.
  • Color System: Easily customize global styling options for text, background, headings, and paragraphs through the Theme Editor.
  • Paragraph Alignment Control: Adjust text alignment (left, center, right, justify) for improved readability and aesthetics.

Developer Tools

  • Vue.js Integration: A lightweight JavaScript framework that allows modular and reusable component-based development for dynamic Shopify content. Improves customer engagement with interactive features.
  • SCSS Integration: Simplifies styling with reusable variables and mixins for scalable and maintainable designs.
  • Centralized Responsive Breakpoints:
    • Defined in a single breakpoints.js file for consistency across SCSS and animations.
    • Seamlessly adapt layouts and animations for mobile, tablet, and desktop devices.

SEO

  • JSON-LD Support: Automatically adds structured data for products, collections, blogs, and pages to improve search engine visibility and support rich snippets.
  • Meta Tag Optimization: Pre-configured meta tags ensure better indexing and sharing across platforms.
  • Social Media Integration:
    • Dynamically generates sameAs links for social platforms based on theme settings (e.g., Facebook, Instagram).
    • Supports multilingual labels and descriptions via en.default.schema.json.
  • Favicon Support: Easily manage your store’s favicon for a polished brand identity.

Note: A detailed feature list will be added in future updates as new capabilities are introduced.
