Setup Instructions

EcoFlow is tailored for entrepreneurs in the wellness, organic, and sustainable lifestyle niches. This guide will walk you through the essential steps to set up and optimize your EcoFlow Shopify theme.

For Merchants

Installation

  1. Purchase the EcoFlow theme from the Shopify Theme Store.
  2. After purchase, click Add to Theme Library in your Shopify admin.
  3. Navigate to Online Store > Themes and locate EcoFlow in your theme library.
  4. Click Actions > Publish to activate EcoFlow as your store’s theme.

Theme Setup

  1. Add Social Media Links:
    • Navigate to Theme Settings > Social Accounts and input your profile URLs (e.g., Instagram, Facebook).
    • Ensure labels and descriptions align with your store’s language and branding.
  2. Set Up Metadata:
    • Go to Theme Settings > Social Meta to define your store’s meta title, description, and sharing image.
    • Localize these settings using translation files (en.default.schema.json) if needed.
  3. Typography and Design:
    1. Open the Theme Editor in Shopify.
    2. Navigate to the Typography and Colors sections.
    3. Adjust:
      • Fonts for headings and body text.
      • Colors for backgrounds, headings, and paragraphs.
      • Font sizes, line heights, and weights for headings (H1–H6) and body text.
  4. Customize Sections:
    • Select a section in the Theme Editor to access its settings panel.
    • Adjust:
      • Layout Width: Toggle between Full Width and Body Width.
      • Spacing: Set local margin and padding overrides for desktop and mobile devices.
      • Section Height: Choose between Auto, Custom (specify pixel height), or Full (viewport height).
      • Background:
        • Upload images and set size, position, and blend modes.
        • Choose mobile-specific backgrounds with separate opacity and blend mode settings.
        • Add video backgrounds for desktop, with fallback images for mobile devices.
      • Borders: Customize top and bottom border styles, widths, and colors.
      • Grid Gaps: Define custom grid gaps to control spacing between elements within sections.
      • Custom CSS Class: Add a class for unique styling or scripting.

Verification and Testing

  • Preview your store using Shopify’s built-in preview tool to ensure all customizations are applied correctly.
  • Verify typography, social media links, metadata, and section settings display as expected.
  • Test responsiveness on mobile, tablet, and desktop devices.
  • Validate structured data using Google’s Rich Results Test to ensure proper SEO implementation.

Advanced Development Notes

Developers can leverage Shopify CLI for local theme development and deploying advanced customizations. Refer to Shopify’s official CLI Documentation for detailed instructions.

Use Vue.js and GSAP to create interactive components and animations, ensuring a dynamic shopping experience.
