Setup Instructions

This page will guide you through the initial steps to set up the EcoFlow Shopify theme.

For Merchants

Installation

  1. Download the theme from your Shopify admin.
  2. Go to Online Store > Themes in your Shopify admin.
  3. Click Upload Theme and select the .zip file you downloaded.
  4. Click Publish to make the theme live.

Theme Setup

  1. Add Social Media Links:
    • Navigate to Theme Settings > Social Accounts and input your profile URLs.
    • Ensure the labels and descriptions match your store’s language for clarity.
  2. Set Up Metadata:
    • Go to Theme Settings > Social Meta to define your store’s meta title, description, and sharing image.
    • Use translation files (en.default.schema.json) to localize these settings if needed.
  3. Typography Setup:
    1. Open the Theme Editor in Shopify.
    2. Navigate to the Typography section.
    3. Adjust:
      • Fonts for headings and body text.
      • Sizes for headings (H1–H6) and body text.
      • Line heights and font weights.

Verification and Testing

  • Preview your store using Shopify’s built-in preview tool.
  • Ensure typography, social media links, and metadata are displayed as expected.
  • Test on mobile, tablet, and desktop devices for responsiveness.

For Developers

Advanced Setup

  1. Define Breakpoints:
    • Create a breakpoints.js file in your project.
    • Add the following content: 
      export const breakpoints = {
  mobile: 768,
  tablet: 1024,
  desktop: 1025,
};
    • Ensure SCSS and GSAP reference this file for consistent breakpoints.

Development Steps

  1. Clone Repository: 
    git clone <repository-url>
  2. Install Dependencies: 
    npm install
bundle install
  3. Build Assets: 
    npm run bundle-js
  4. Deploy to Shopify: 
    shopify theme push
